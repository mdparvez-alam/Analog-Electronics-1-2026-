## Real Application
The 1N4007 diode is commonly used in AC to DC power supply circuits. 
It works as a rectifier diode, allowing current to flow in only one direction.
This is necessary to convert alternating current (AC) into direct current (DC)
in adapters, chargers, and power supply units.
